# 2D Object Clearing Simulation (Matter.js)

A simple browser-based project I built in 2020 while learning JavaScript.   

## Overview 

This project demonstrates a basic 2D interaction system using Matter.js. The player controls a yellow block that can move around the screen and push other objects away, effectively clearing the space.

## Features

* Player-controlled movement
* Collision handling  
* Object removal by pushing off-screen

## Tech Stack

* JavaScript
* HTML / CSS 
* Matter.js

## Purpose

This was one of my early experiments with:

* Game mechanics

## Controls

* Move your mouse to control the yellow block

## Note

This is an early learning project and not a polished game. It represents my first steps into interactive simulations and physics-based movement.
